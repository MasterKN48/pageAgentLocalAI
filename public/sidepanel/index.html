Page Agent

Injects the AI agent into the active tab. Give it natural language commands to interact with the page.

How it works

  1. Load the model (one-time, cached after first load)
  2. Click "Activate" on any page
  3. Use the chat panel to give commands
  4. The AI reads the page DOM and performs actions

All processing happens locally on your device. No data is sent to any server.
